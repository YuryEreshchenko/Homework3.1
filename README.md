# RGB LED Light Stick

This Arduino project controls a **common-anode RGB LED** using a single push button.  
Each short or long press changes the LED’s behavior between multiple lighting modes.

---

## Features
- **Short press** → Changes LED color  
- **Long press** → Switches between light modes  

---

## Modes
| Mode | Name        | Description |
|------|--------------|--------------|
| 0 | **Static** | Displays the selected color constantly. |
| 1 | **Blink** | Toggles the LED on/off at a steady pace. |
| 2 | **Fade** | Gradually fades brightness in and out (breathing effect). |
| 3 | **Strobe** | Rapid flashes of the chosen color for a strobe-light effect. |
